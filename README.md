# The Pucchi
Every NCERT chapter. Interactive. Free. Forever.

Live site: https://thepucchi.github.io

An open-source interactive textbook for Indian students — built chapter by chapter, subject by subject, class by class. Each chapter is a standalone HTML page with animations and simulations.

## Structure
```
index.html                         ← Homepage
class-11.html                      ← Class 11 subjects
class-11-physics.html              ← Class 11 Physics chapters
class-11-physics-ch4.html          ← Chapter 4: Motion in a Plane (live)
```

## Contributing
Each chapter is a single self-contained HTML file. No build step. No dependencies. Open any file in a browser and it works.
